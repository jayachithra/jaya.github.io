Simple C# MCP Server Setup

Build a quick MCP Server in C# and inspect it using the inspector tool.

Date: 2025-08-06

Introduction

Sharing my learning on creating a simple MCP Server in C#, locally running it and inspecting it using the inspector tool.

Step-by-Step Guide

  1. Create a simple MCP server for Weather API following the steps given here. Or clone the repository here.
  2. Install the inspector using the command npx @modelcontextprotocol/inspector. Prerequisite - Node.js: ^22.7.5
  3. Run the command npx @modelcontextprotocol/inspector dotnet run from within the mcp server folder. This will open the UI for inspector in browser.
  4. Click on the connect button, go to Tools option from the top panel and click on "List Tools". You should see the list of tools provided by the MCP Server.
